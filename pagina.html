  • git init - inicializa um novo repositorio git
  • git add - adiciona arquivos
  • git status - mostra o estado atual do repositório
  • git commit - salvando as mudanças que estão na staging area
  • git push - envia os commits locais para o repositório remoto
  • git log - mostra o histórico de commits do repositório
  • git log --oneline - mostra o histórico de commits em formato resumido
  • git branch - lista todas as branches
  • git checkout - troca para outra branch ou commit
  • git checkout -b - cria uma nova branch e já muda para ela
